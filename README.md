## 项目介绍
    该项目实现了疫情地图的可视化，采用前后端分离的模式，前端采用echarts绘图，Ajax传输数据，后端是servlet+Java，数据来自丁香医生。
## 如何构建和运行
#### 运行环境：
    eclipse、Jdk1.8、tomcat9.0

#### 运行方法：
    下载InfectStatisticWeb项目后将servlet文件夹导入eclipse，展开Servlet/WebContent，右键点击main.html=>运行方式=>Run on Server即可运行（前提是eclipse配置好tomcat）。
